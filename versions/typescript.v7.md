# [typescript 7.0 新特性](https://baijiahao.baidu.com/s?id=1857486497365571632&wfr=spider&for=pc)

1. 在 VS Code 扩展商店搜索并安装 `TypeScript Native Preview` 官方扩展。

2. 在项目的 `tsconfig.json` 中添加配置，启用 Go 编译器。

3. 重启 VS Code，即可体验即时提示与快速编译。

```sh
npm install -g typescript@7.0

tsgo build
```

几乎与 `tsc` 用法一致，原有习惯完全保留。
